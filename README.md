Funktsionaalsused:
  1. Kella värvi muutmine
  2. Saab vahetada taustapilti vastavalt neljale erinevale aastaajale
  3. Taustapildi muutudes hakkab mängima ka heli
  4. Nupp taustal mängiva heli vaigistamiseks
  5. Nupp heli taastamiseks
